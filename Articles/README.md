# Articles Directory

Place your PDF full-text articles in this directory for data extraction.

## Usage

1. Add PDF files to this directory
2. Run the extraction script: `python gemini_extractor.py`
3. The script will process all PDF files and extract data according to your template

## Notes

- PDF files are excluded from git tracking (see `.gitignore`)
- Subdirectories like `Reviews/` are also supported
- Make sure PDFs are readable and not password-protected
